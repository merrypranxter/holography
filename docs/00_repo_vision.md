## What this repo should expose

Visual primitives implied by holography:

- **area controls information**
- **boundary generates bulk**
- **radial depth = scale / resolution**
- **light propagates along null structures**
- **entanglement appears as minimal bridges / shared geometry**
- **nonlocality means distant surface patterns co-control interior form**

That is the usable core.

The source material supports exactly that framing: black-hole entropy scales with **horizon area**, not volume; the covariant entropy bound uses **light-sheets** with nonpositive expansion; AdS/CFT maps **boundary data to bulk fields**; and holographic entanglement entropy is computed by **minimal-area bulk surfaces**.

## Core encoding philosophy

Every physics concept goes through this pipeline:

```
physics idea
  → geometry
  → field behavior
  → information-density rule
  → rendering consequence
  → controllable parameters
```

Do not stop at "physics idea."
Do not stop at "geometry."
Go all the way to controllable parameters.

## The central anti-fake test

Ask these questions of every scene:

1. Is bulk structure caused by a boundary/source/screen law?
2. Does depth change scale or recoverability?
3. Are light-sheet and screen scenes directional and terminating?
4. Are entanglement bridges driven by relation, not ornament?
5. Are black-hole interiors losing readability?
6. Is near-boundary structure separated into source/divergence/finite response?
7. Are hostile geometries allowed to fail clean local reconstruction?

If not, the scene is "holography flavored," not holography translated.

## Layer architecture

Every scene should respect four layers:

### Layer A — scene topology
SDF bodies, boundaries, shells, horizons, membranes, wedge volumes, screens.

### Layer B — holographic fields
Boundary source fields, response fields, entanglement weights, null-projection density, renormalization terms, precursor channels, cosmological pressure.

### Layer C — transport / reconstruction
Local smearing, nonlocal precursor injection, light-sheet growth, coarse-graining, horizon scrambling, RG filtering.

### Layer D — image formation
Raymarch shading, glow, spectral lensing, post-process diffusion, chromatic anomaly bands, debug overlays.
